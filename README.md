# Flappy Bird Game

A simple full-screen Flappy Bird game made using HTML, CSS, and JavaScript.

## How to Play

* Press **Space** or **Arrow Up** to fly.
* On mobile, tap the screen to fly.
* Avoid the pipes and the ground.
* Try to beat your best score.
